Canoe Portage

Assignment Overview:
This assignment involves coding and testing of a program that uses Python Arithmetic.

The basic design of your program prompts the user for input, performs arithmetic on that information, and then displays the results. The program, coding wise is very straight forward, but it may take some time to understand the mathematical conversions.


Program Specifications:
* Your program will prompt the user for an integer value representing a distance in rods. You will reprint that value along with that value converted to the following values:
  - meters
  - feet
  - miles
  - furlongs
  - the time in minutes to walk that distance

You can find these measurements on the web, but here are the conversions I used:
  - 1 rod = 5.0292 meters
  - 1 furlong = 40 rods
  - 1 mile = 1609.34 meters
  - 1 foot = 0.3048 meters
  - average walking speed is 3.1 miles per hour
