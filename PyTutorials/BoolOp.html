Boolean Operators

In the following tutorials, the three boolean/Logical operators we are going to take a look at are AND, OR, and NOT.

The AND operator:

Looking at the different comparisons that we have provided, let’s see what is returned:

>>> #In this example, both comparisons are correct/True and thus our end result is True.
>>> print(123 == 123 and 10000 < 124532)
True

>>> #Here, only one comparison holds True
>>> print(768 != 442 and 1344987 <= 10)
False

>>> #What happens if both statements are False?
>>> print(903 >= 738292 and 6 > 1024)
False


When it comes to the and operator, both statements have to be True for the end result to return True. If even one is false, the whole statement turns False.

The OR Operator

The OR comparison is a little more lenient than the AND; only one condition must hold true in order for the statement to return True

>>> #If both are True
>>> print(165 == 165 or 12365 >= 547)
True

>>> #If one is True
>>> print(76 <= 809 or 678 != 678)
True

>>> #If neither are True
>>> print(90 < 65 or 78 > 345)
False

Take this real life scenario:

As students yourselves, this example will hopefully clear things up: “You have completed your homework and it is time for bed” Here, because both statements check out/are true, you go to bed. However if I were to change the statement to: “You have not completed your homework or it is not time for bed.” In this scenario, only one of these statements needs to check out true for you to not go to bed.

Hopefully, now you have a better understanding on how these statements work! Let’s look at one last operator.

The NOT Operator

Put simply, the not operator negates the value of the conditional

>>> print(not 2 == 2)
False
>>> print(not 34 >= 65)
True

As you can see, when adding the not keyword before the conditional flips the value that it was before. For example, in the first print statement, 2 == 2 would originally be True but because of the not operator that was added in front, the value True now becomes False.

-----------------------------------------

You can take these simple operands to the next level by using more than one of them in a single statement. For instance:

>>> print(not 34 > 343 and 455 == 354 or 87 <= 87)

Can you figure out what this will return? Try it out on your IDE to check your answer!
