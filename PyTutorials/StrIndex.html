String Indexing

Strings consist of a grouping of characters; they can be letters, numbers, and even special characters. Each character in a string also has its own position/index. Think of it this way: all of the characters in the string are standing in a line, the first one has the first position, while the last one has the last position number. However, one key difference to note her is that the first character index is not 1 but instead 0.

>>> word = "Python"

The first letter in this example is ‘P’ and it’s index is 0.

The second letter ‘y’ has an index of 1.

This pattern continues to the end of the string (last letter ‘n’ is at 5th index).

In Python, we can access certain positions of our string using indexes.

>>> print(word[0])
P
>>> phrase = "This is a string with spaces!"
>>> print(phrase[5])
i

You may have noticed that in this example, the space was also assigned a position.

So far so good! But what if we had a very long string and were told to get the very last character. Fortunately, Python gives us a way to more conveniently access either side of a string

>>> print(phrase[-1])
!

Using the negative sign before the number when indexing tells Python to start counting from the left last character. In our case, the last character is an exclamation point ‘!’.

Now that you have an idea of how to retrieve a single character, can you think of how to get a certain group of characters?

The fancy term for using indices to get a group of characters rather than only one is slicing.

>>> thought = "I love to code!"
>>> print(thought[2:6])
love
>>> print(thought[-5:-1])
code

The basic syntax for this is as follows: string[start:end + 1])

You may have noticed in the above example that when using this method, Python includes the character at the start index and ends at the character before what is at the end index.

Another option is to actually ask for the starting index of a substring using the index() method.

>>> print(thought.index("code"))
10
