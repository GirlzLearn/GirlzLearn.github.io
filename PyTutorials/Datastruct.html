Data Structures

Lists

What are lists, how are they created, and what are their functions?

Recall the four main data types: strings, integers, floats, and booleans. Similar to data types, there are also what we call data structures. Lists is one of the most frequently used and versatile of the four data structures (lists, tuples, sets, dictionaries).

Creating a List:

As the name says, a list is composed of items/elements. In python, we wrap our list in square brackets ‘[]’ and separate each item with a comma. There is no limit on the number of items in a list and they can include any of the data types.

#empty list
my_list = []

#list of strings
my_list = [123, 234, 654, 760]

#list of integers
my_list = ["sun", "moon", "clouds"]

#list of mixed data-types
my_list = [1, "hi", 3.14, True]

We can also include lists inside of the list. These are called nested lists.

#nested list
my_list = ["happy", ["a", "b", "c"], [1, 2, 3]]

List Indexing

Similar to string indexing, we can use indexing to access a given element. Remember, the first indices is 0.

my_list = ["Python", "rocks", [2, 4, 6, 8]]
print(my_list[0])

Slicing Lists

Again, the format for slicing lists works the same way as slicing strings. The only difference is that the return type is a list.

my_list = ["g", "i", "r", "l", "z", "l", "e", "a", "r", "n"]

print(my_list[0:4])
['g', 'i', 'r', 'l']

#elements to end
print(my_list[5:])
['l', 'e', 'a', 'r', 'n']

#elements beginning to end
print(my_list[:])
['g', 'i', 'r', 'l', 'z', 'l', 'e', 'a', 'r', 'n']

Unlike strings, lists are mutable, meaning you can change the elements/items in a list

Changing Elements in a list

Changing/replacing elements in a list is actually quite simple and can be done with indexing and an ‘=’ sign:

odd_change = [1, 3, 5, 7]
odd_change[0] = 0
print(odd_change)
[0, 3, 5, 7]

#You can also replace a range of items
odd_change[1:] = [2, 4, 6]
print(odd_change)
[0, 2, 4, 6]

Append/Extend - Adding to the end of a list

There are two ways to add to the end of a list: append() and extend(). While append only allows you to add one item at a time, extend lets you add as many items as you want to the list.

even = [0, 2, 4, 6]
even.append(8)
print(even)
[0, 2, 4, 6, 8]

even.append([10, 12])
print(even)
[0, 2, 4, 6, 8, [10, 12]]

As you can see, when you use append() and try to add more than one item, it is treated as a nested list rather than separate elements.

Let’s use extend() and see what we get:

even = [0, 2, 4, 6]
even.extend([8, 10, 12])
print(even)

[0, 2, 4, 6, 8, 10, 12]

Math with Lists

Similar to strings, we can use the ‘+’ to concatenate two lists

List1 = [1, 2, 3]
List2 = [4, 5, 6]

print(list1 + list2)

[1, 2, 3, 4, 5, 6]

You can also insert items into the list at desired indices using insert():

odd = [1, 3, 11]
odd.insert(2, 5)

print(odd)

[1, 3, 5, 9]

Format of insert(): list.insert(<index>, <new-item>)

You can also insert multiple items into a list using an empty slice.

odd[3:3] = [7, 9]
print(odd)

[1, 3, 5, 7, 9, 11]

You can also delete elements from a list using del, pop(), remove(), and clear()

del is used to delete the item/items at a specific index:

nums = [0, 1, 2, 3, 4]


del nums[0]
print(nums)
[1, 2, 3, 4]


del nums[1:3]
print(nums)

[1, 4]

pop() removes the element at the specified index. Unlike del, pop() can only delete a single item and not a range of items. pop() can also return the item deleted.

nums = [10, 20, 30, 40]
print(nums.pop(2))

print(nums)

30

[10, 20, 40]

remove() deletes the first instance of an element by using the given value rather than index

nums = [100, 300, 500, 100]
nums.remove(100)
print(nums)

[300, 500, 100]

Finally, you can use clear() to empty the list

nums = [300, 500, 100]
nums.clear()
print(nums)

List Comprehensions: A crisp and dynamic way to create lists

List comprehension is simply a way to create a new list using for loops and possibly if statements.

Format: listname = [<expression>]

Let’s start with for loops:

pows = [x**2 for x in range(1, 11)]
print(pows)

[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

From this example, you can probably see that this example is a concise way to write something like this:

pows = []
for x in range(1, 11):
        pows.append(x**2)
print(pows)

[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

You can make the list more precise to your needs with an additional if statement.

pows = [x**2 for x in range(11) if x > 4]
print(pows)

[25, 36, 49, 64, 81, 100]

Can you see the power of lists and list comprehensions? Pretty cool, right? I highly encourage you to play around with lists outside of this tutorial. I have only covered a few of the main methods that lists provide so be sure to check out more. You can visit Python 3 - List Documentation.
