Arithmetic Operations

Let’s finally start to have the computer do work for us! In this tutorial, we’ll turn Python into a mini calculator…

Assuming you are familiar with basic arithmetic such as addition, subtraction, multiplication, and division, implementing these skills into Python is fairly simple. If you recall from a previous tutorial, the print command is used to directly receive some output from the computer upon giving some sort of input.

print(12 + 32)
print(434 - 233)
print(90 * 2)
print(56 / 7)

^ Try it out and see what you get!

Some more division…

Hopefully, you noticed that when you ran the last operation print(56 / 7), the returned value was 8.0 rather than 8 and maybe you also recognized that 8.0 is a float, not an integer. If, however, you did want an integer to be the return value, you would replace the ‘/’ with ‘//’

print(56 // 7)

Another operator you could use is the ‘%’ or modulus (mod) symbol. Mod is applied when you are looking to find the remainder value when dividing two numbers.

print(57 % 7)

If you ran this, you’ll see that the type of the returned value is an integer.

Now that you know a couple basic operations, feel free to play around on your IDE!
