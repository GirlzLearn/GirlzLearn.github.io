Loops

Loops are used to iterate through a block of code based on a given expression

While Loops

In while loops, the code in the loop runs if the condition is true.

Let’s take a look at an example:

n = 0
while n <= 6:
 print(n)
 n += 1

In this example, note that  n += 1  is just a more concise way of writing n = n + 1

Format:

while <expression>:

        <code to run if expression is true>

NOTE: make sure there is a mechanism to end to loop. Otherwise, you could end up with an infinite loop (it won’t stop running on your console) For example, if instead of using n+=1, you say n-=1, you will be stuck in an infinite loop since n will always be less than 6.

Break & Continue

Sometimes, you want to exit the loop, even if the condition still holds true. To achieve this, we use the break command.

n = 0
while n <= 6:
        print(n)
        if == 3:
                break
        n += 1

Other times, you may want to skip an iteration and continue on with the loop. You can use the continue command for this.

n = 0
while n <= 6:
        print(n)
        if == 3:
                continue
        n += 1

Feel free to play around with these two commands and see what you get!

Similar to the way an if statement works, when the while condition is false, you can use an else statement to run something else.

n = 0
while n < 6:
        print(n)
        n += 1
else:
     print("n is greater than 6")

For Loops

For loops are slightly different. They iterate through a sequence (this can be a string, list, tuple, dictionary, etc). The program will end once the computer finishes iterating through the sequence or there is a break in the loop.

If you are not comfortable with data structures yet, I recommend you take a look at the lists, tuples, and dictionaries tutorials.

names = ["sally", "andy", "bob"]
for name in names:
        print(name.capitalize())

Format:

for <iterator> in <iterable>:

        <code to run while sequencing through iterable>

In a for loop, the iterator is just a placeholder while you run the loop to go through the iterable (whatever string, list, tuple, etc. that you want to go through). As you can see in the example above, you do not need to initialize an iterator beforehand.

Here is another example:

even = []
for i in range(10):
        if i % 2 == 0:
                even.append(i)

You can also add if statements or even the break and continue commands we looked at above to add more context to your for loops.
