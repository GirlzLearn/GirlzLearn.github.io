What Is An IDE

So how do programmers code?

What is an IDE? Put simply, an IDE (Integrated Development Environment) is where programmers write their code and get the final outcome. IDEs allow programmers to consolidate separate pieces of their code while writing their programs. Additionally, IDEs increase producer productivity through their assistance of several features and tools. IDEs range from basic platforms with only a couple of these features to project ready environments with a vast variety of tools.

Some basic features of an IDE include the following:

  1. Text Editor/Syntax-Highlighting
  2. Auto-Completion
  3. Debugger
  4. Compiler
  5. Built-In Plugins and Packages

What is Text Editor:

Text Editors are pretty much the base of the IDE. This is the feature that actually allows users to play with the code. Text Editors typically have some visual aspects that make coding more comfortable, convenient, and fun! The most basic of these is language-specific syntax highlighting. In other words, if the computer runs across a word or phrase that’s reserved for the specific language (Python in this case), then it will automatically be highlighted in a different color so it stands out.

What is Auto-Completion:

Well, as the name suggests, this is a feature of IDEs that can pretty much predict what you are writing and give suggestions to complete the code. This feature saves tons of time and reduces chances of having any typos in your code.

What is a Debugger:

As codes get increasingly complex, it can sometimes be difficult to understand what went wrong if the program doesn’t run. In these cases, debugging tools are there to assist you in identifying and fixing the errors in their code. These tools come in handy more often than you would expect so it is pretty important to get a handle on them.

What is a Compiler:

The compiler is a key component of an IDE. This is where the programming language that you used to code something is translated into machine language that the computer can understand (ex. Binary code). Fortunately, Python is an interpreted language and not compiled like Java, C, C++. Although compilation does occur when adding the .py extension to your file, there is no formal step like in other languages.

For more information on how Python works behind the scenes, visit (How does Python work? A simple explanation of how Python code) There is a good amount of information on the site that the author provides if you are interested.

Built in Plugins/Programs:

For more advanced IDEs, there are numerous additional plugins, or extensions, that further enhance your experience. For this reason, fully understanding what your IDE can offer will likely save a lot of time and frustration.
