Installing Python

Python is offered across numerous platforms or IDEs but there are definitely a few that receive the best reviews and that I personally love.  

IDLE

Pros:

This IDE is likely best suited for programmers who are just starting out mainly due to its nice simplicity. Some features that this platform has includes syntax highlighting. IDLE also has tremendously easy set up and you can get going right after downloading the platform. Additionally, IDLE is a free platform and does not take up much space on your desktop

Cons:

Once you get going and your code gets more complex, I would not recommend this platform as much because there aren’t many proper tools for debugging and testing code on IDLE. Also, because of its functionality, if there is a bug in your code, the code breaks before you can continue working on it, making it difficult to work with for larger projects.

To download IDLE…  

Make sure when downloading that you select the latest version

  • As of 2020, that is Python 3.8, the version may change but as long as you are above python 3.5, you are good.

Mac OS X version:

  • https://www.python.org/downloads/

Windows version:

  • https://www.python.org/downloads/windows/

Linux version:

  • https://www.python.org/downloads/source/

Atom

Pros:

Atom is a great cross platform text and source code editor developed by GitHub. Many features of an IDE are showcased on Atom including auto completion, syntax highlighting, and installation of other packages. Multiple panes can be opened and you are also allowed to customize the editor to satisfy your liking. Additionally, as you begin to collaborate on your code and use GitHub, Atom is very convenient.

Cons:

Because Atom is only a code editor and not specifically designed for python, there is no built in console to run your scripts as a beginner and would have to use the command line to do so. Also, you will see debugging tools are not the best on Atom.

To download Atom:

  • https://atom.io/

Repl

Pros:

Since Repl is a browser application, there is no need to download anything. For this reason, you can work on your programs from public computers. Syntax highlighting and auto-completion suggestions above other features are also included. Additionally, unlike most web apps, upon logging into your account, all your progress is automatically saved. Repl also allows for retrieving files from GitHub, which is nice.

Cons:

Because Repl is online and depends on fast internet, if your access to the internet is not the best, running scripts can grow to be quite daunting.

Getting Started with Repl

  • https://repl.it/

PyCharm

Pros:

There are many plugins offered as well as other packages that you can download. Debugging tools in Pycharm are probably the best that I’ve ever seen. Syntax highlighting and auto-completion is also seen on Pycharm. There is also a built in Python Console to run your scripts through, as well as a built in terminal.

Cons:

The IDE can seem bulky and maybe too much to handle, especially for beginners. For this reason, I would not recommend Pycharm for complete beginners in this language, unless you have someone to show you around.

To Download PyCharm…

  • Although there is a professional download option which you would have to pay for, the community version works just fine.
  • https://www.jetbrains.com/pycharm/download/#section=mac

As mentioned previously, all of these IDEs have their strengths and weaknesses and you are free to choose your favorite. For information on how to install any of these IDEs, simply visit the links provided. I highly recommend that you practice running your scripts on your command line as it is important to have a basic understanding of the Linux Shell. For information on what the terminal is and how to do this, visit my bash tutorial page.
