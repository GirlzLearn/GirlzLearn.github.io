IF, ELSE and ELIF Statements

In the previous tutorials, we have looked at the different conditional statements and the True or False aspect of it all. Now, let’s put that knowledge to use and actually do something based on that boolean value.

The IF and ELSE blocks are used to accomplish this:

x = 10
if x < 50:
   print("Your number is less than 50")
else:
   print("Your number is greater than 50")
print("Done")

If you run this, you will see that the return answer is “Your number is less than 50” followed by “Done”. Looking at the if statement, we see a conditional ‘x < 50’. Here, Python traces back to when x was declared and remembers that it is equal to 10. Now computing that x (10) is in fact less than 50, the statement is True. Python then runs whatever is inside of the if block (which in this case is our print statement) and then skips the else statement and goes down to print “Done”.

   

It is important to recognize that only one of the statements will run (either the if or the else). If the condition is true, the body of the if block will run and when the condition is false, the else block will run.

Now that you hopefully understand how if and else blocks work, let’s go back to our example. What if we change the value of x to 50 and run the code:

x = 50
if x < 50:
   print("Your number is less than 50")
else:
   print("Your number is greater than 50")
print("Done")

Return:

Your number is greater than 50

Done

There’s a bug in our code! x is equal 50 not greater. Fortunately, in Python, there’s another statement we can add in our code to make this work out:

x = 50
if x < 50:
   print("Your number is less than 50")
elif x == 50:
   print("Your number is equal to 50")
else:
   print("Your number is greater than 50")
print("Done")

Return:

Your number is equal to 50

Done

Keep in mind that only one of these statements will run which one that may be is based on the condition in the if block and later in the elif statements. Also, there can only be one if statement that starts it off. You can, however, have as many elif statements as you feel necessary but end with only one else statement.

Think of these statements like a sandwich. You only put one bread on top (that serves as the if), as many layers in the middle, which help add more to your sandwich (elif statements), and finally, one slice of bread on the bottom to finish it off (else).

Also, you may have noticed that while the if and elif statements have conditions, there is nothing for the else statement. This is because the else block serves as a sort of “if all else fails then run what’s in the else block”.
